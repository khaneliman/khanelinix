# Progress Tracking Play

Use for long-running game work across multiple agent turns.

## progress.md

If `progress.md` exists, read it first. Confirm original user prompt is recorded
at top:

```text
Original prompt: <prompt>
```

If missing, create it before appending updates.

## What To Record

- implemented features
- verified controls/scenarios
- screenshots or artifacts inspected
- console errors fixed
- remaining TODOs
- suggestions for next agent

Keep updates concise and factual.
