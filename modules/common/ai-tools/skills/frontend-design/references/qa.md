# Frontend QA Play

Use before final response for UI work.

## Functional Checks

- Primary workflow works end-to-end.
- All visible controls respond and update state.
- Error/empty/loading states render when relevant.
- Keyboard/focus behavior works for important controls.

## Visual Checks

- Desktop and mobile viewport screenshots inspected.
- Text does not overlap, clip, or overflow containers.
- Important first-viewport content visible.
- Dense realistic state still fits.
- Contrast and hierarchy support scanning.
- Motion does not obscure or destabilize content.
- Visual assets load and are not dark/blurred/cropped when inspection matters.

## Report

State checks run, screenshots or viewports inspected, and remaining visual risks.
