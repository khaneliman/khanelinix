## Codex Delegation

Delegate bounded multi-file discovery, noisy probes and tests, and parallel fact
slices to the smallest configured worker. Keep planning, integration, and final
judgment in main thread. Pass only task, paths, constraints, allowed tool or
skill lane, and exit criteria; consume an evidence packet.

Use `git-toolkit` change-stack mode for multi-commit or PR-stack boundaries.

## Retry Circuit Breaker

After two failures from the same command family, endpoint, test target, or patch
shape, stop repeating it. State both attempts and signals, then change lane.
Resume only with a materially different command, narrower scope, or new
evidence.
